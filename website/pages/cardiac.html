Cardiac Arrhythmia Detection

Neuromorphic rhythm classification in 928 bytes of RAM. Detects tachycardia, bradycardia, and irregular rhythm using delay-line coincidence detection.

Key weakness: The Gaussian tuning cannot reject in-band frequency noise. Real ECG signals require dedicated preprocessing (bandpass filter, wavelet denoising) before CricketBrain processing.

How It Works

ECG Signal ──> R-Peak Extraction ──> RR Intervals ──> Frequency Mapping
                                                             │
                                              CricketBrain (5N/6S, 928 bytes)
                                              Gaussian tuning at QRS freq
                                              Coincidence detection gate
                                                             │
                                                      Spike Output
                                                             │
                                                    RR Interval Tracker
                                                    (sliding window, 8 beats)
                                                             │
                                              ┌──────────────┼──────────────┐
                                          >100 BPM      60-100 BPM      <60 BPM
                                         Tachycardia   Normal Sinus   Bradycardia
                                                    CV > 0.3 → Irregular

Quick Start

Terminal 
# Clone and build
git clone https://github.com/BEKO2210/cricket-brain.git
cd cricket-brain/use_cases/01_cardiac_arrhythmia

# Run synthetic demo
cargo run --release

# Classify CSV data
cargo run --release -- --csv data/processed/sample_record.csv

# Run all benchmarks
cargo run --release --example cardiac_sdt
cargo run --release --example cardiac_latency
cargo run --release --example cardiac_stress
